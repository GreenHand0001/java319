# 停车管理系统 java319

本项目是一个基于SpringBoot+Vue+MySQL的停车管理系统（车辆管理系统），采用Java语言开发，实现了前后端分离的Web网站。

## 技术栈

### 系统架构
- B/S架构
- MVC模式

### 系统环境
- Windows/Mac

### 开发环境
- IDE：IDEA
- JDK版本：1.8
- Node.js版本：16
- 构建工具：Maven
- 数据库：MySQL
- 数据库管理工具：navicat

### 技术选型
- 后端：Java、SpringBoot、MyBatis
- 前端：Vue
- 数据库：MySQL

## 部署步骤

### 数据库搭建
1. 新建MySQL数据库。
2. 执行提供的SQL文件，完成数据库初始化。

1. 进入项目目录。
2. 运行`npm install`安装依赖。
3. 运行`npm run dev`快速启动前端服务。

1. 配置`application.yml`文件。
2. 启动SpringBoot项目。
